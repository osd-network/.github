# Contributing

Thank you for your interest in contributing to **osd-network**.

## How to Contribute

1. **Fork** the repository and create a branch from `main`.
2. Follow the existing folder structure (`1ciclo/`, `2ciclo/`, `3ciclo/`, etc.).
3. All content must reference **official National normative sources** (e.g., PT: DGE, DR, DGES)
4. Open a **Pull Request** with a clear description of what was added or changed.

## Standards

- Cite the specific *Despacho*, *Portaria* or *Decreto-Lei* for every curricular claim.
- No copyrighted manual content - reference documents only.
- Keep markdown files clean and consistently formatted.

## Issues

Use [GitHub Issues](../../issues) to report errors, outdated references, or missing content.
